Flutter部分插件后续持续更新～

发表于 2019-01-25 | 分类于 Flutter

Flutter插件 部分插件

这里的插件一部分是我在开发过程中使用过的，一部分是用于留着以后可能会需要使用的～

名称 解释
GPS sensor GPS sensor
Camera 相机模块
Facebook 使用Facebook的第三方登录等API
Shared Preferences 轻量级存储类，以键值对的形式保存设置，属性和数据
SQLite 一款轻量级的关系型数据库，类似SQLite
Firebase Cloud Messaging Google提供的一个云服务，应用一般可以接入它来实现消息推送等功能（大天朝可自动忽略）
pull_to_refresh 上拉加载和下拉刷新的组件
dio 数据请求
json_serializable json数据解析
flutter_spinkit 一组加载指示器
intro_slider 一个插件，可以帮助您制作介绍屏幕，以显示您的应用程序的主要功能。您可以更改图像，按钮，文本样式，颜色等内容。创建介绍屏幕从未如此简单快捷
intl 提供国际化和本地化设置
scoped_model 状态管理
flutter_crashlytics 可启用Crashlytics报告
event_bus 使用Dart Streams进行解耦应用程序的简单事件总线
flutter_alipay 支付宝登录
fluwx 微信登录
flutter_staggered_grid_view 瀑布流布局
flutter_dropdown_menu 下拉菜单
BannerView swiper轮播图
flutter_swiper swiper轮播图
flutter_cupertino_date_picker 日期选择器
image_picker 单图片选择器
multi_image_picker 多图片选择器
connectivity 检测网络状态
path_provider 用于查找文件系统上常用的位置
package_info 用于查询有关应用程序包的信息
photo_view 图片photo view
amap_location 高德定位插件 IOS的返回信息部分属性缺少为null 的判断，详情看Issues
simple_permissions 获取权限 swift写的，项目需要支持swift
cached_network_image 缓存网络图片
lpinyin 汉字转拼音
flutter_webview_plugin webview 插件，展示H5页面后，安卓手机点击或者双击屏幕再按物理返回键，会展示Initchild，而没有直接返回上一页，有大佬知道原因，求告知
qr_flutter 可通过Widget或自定义简单快速的进行 QR码 渲染
barcode_flutter 可通过Widget或自定义简单快速的进行 条形码 渲染
flutter_svg 在Flutter Widget上绘制SVG
disk_lru_cache 缓存问题这个插件我还没试过，留着了解一下
